Easy-Documentation

== Description ==

Easy-Documentation is plugin designed for developers who want to make fast documentation for their customers. 
It gives you tutorial how to manage content on your website in just few steps.  
We are trying to satisfy developers in 100% so that is why we are waiting for your opinion how to update our plugin.

1.Clarity of informations - very clear interface

We are trying to lead people to manage their sites in the best way so we are sharing our experience in building WordPress sites.  
We are writing tutorial how to manage your site in the most clear version for beginners with the most readable 
interface how we were able to achive.

2.Customization

You can customize by yourself documentation so as to people working on your site will always be able to manage it on their own with your tutorial.
You can customize it with our predefined flexible headers easily.

3.Actively developed

Easy-Documentation is actively developed with new features all the time. 




== Installation ==

HOW TO MANAGE EASY-DOCUMENTATION :

1. Upload plugin folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Easy-Documentation plugin should be accessable from your „Dashboard”
4. Easy-Documentation contains two submenus - Documentation (where your documentation content is displayed) and Settings. In Settings submenu you can find image guide how to add entries (Screenshot1). You can change documentation title (which is displayed on top of Documentation subpage) as well.
