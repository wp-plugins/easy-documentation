Easy-Documentation

== Description ==

Want to provide documentation for your website via Wordpress Dashboard? Then Easy-Documentation plugin is made for you. 
With build-in editor, you can create step-by-step instruction for less experienced users i.e. how to add new post or how to edit any content of your website.

Create list of potential problems. Show how to solve them with text instructions and images.

Solutions are accessible with only few clicks. Find Documentation icon in your Dashboard and select item from the list.

No more documentation in external files!


== Installation ==

HOW TO MANAGE EASY-DOCUMENTATION :

1. Upload plugin folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Easy-Documentation plugin should be accessable from your „Dashboard”
4. Easy-Documentation contains two submenus - Documentation (where the actual documentation is displayed) and Settings. Settings submenu is the place where documentation is created. Look at instruction image to understand how to write your own solutions.
